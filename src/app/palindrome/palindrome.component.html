What's a palindrome?

A palindrome is a word, number, phrase, or other sequence of characters which reads the same backward as forward, such as madam or racecar or the number 10801.

The main purposose of this excerice's find the largest palindrome in a given string.

i.e: string 'bmb 444414444 4949 stats' should return the string '444414444'.
